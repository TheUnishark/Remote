ProxyBlocker Terms

Definitions

(1) Player: Represents a user account in the video-game Broke Protocol.
(2) Proxy: Represents one or multiple intermediate server between the user and the Broke Protocol server. It includes but is not limited to public VPN services.
(3) IP: Represents the account's public Internet Protocol address such as an IPv4 or IPv6.
(4) ProxyBlocker API: Represents the service used to send players IP and receive validation information.


Service Description

ProxyBlocker is a service by TheUnishark that aims to prevent players from using proxies to play in a Broke Protocol server.


User Data

Conditions:

This service will reject a connection to the server if they do not accept the terms.

If a player's IP is unable to be verified the player will be allowed to play the Broke Protocol server and will tell the administration of the server this player is playing without verification.

A player can skip all verifications if their account have the permission pb.bypass.


Usage:

This service sends the player's IP to third parties to verify if it's detected as a proxy. This IP is saved in a database to make the process faster.

This service will maintain the confidentiality of the player's IP address and will not share it with any third-party organizations or individuals unless required by law.

This service will ensure the security of the player's IP address and will take all necessary measures to prevent unauthorized access or misuse.

This service will comply with all applicable laws and regulations regarding the handling and processing of user data.

Any use of the ProxyBlocker API without authorization will result in legal consequences.


Third parties:

This service uses Google Cloud services to host the ProxyBlocker API and send the IP address to IPQualityScore.

This service sends players IP to IPQualityScore to verify them. The only collected information is the proxy field result.

This service uses Free MySQL Hosting services to save players IP.

RGPD

The players can see their information with the in-game command /pb-get.
The players can opt-out of this service anytime by using the in-game command /pb-remove or by contacting TheUnishark with their IP to remove it from the database (contact information below).
